Installation

  • Install Numo::NArray

  • Install GSL - GNU Scientific Library - version 2.1

  • Install Numo::GSL

    $ git clone git://github.com/masa16/numo-gsl.git
$ cd numo-gsl
$ rake build
$ gem install pkg/numo-gsl-*.gem
