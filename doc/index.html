Installation

  • Install Numo::NArray

  • Install GSL - GNU Scientific Library - version 2.1

  • Install Numo::GSL shell $ git clone git://github.com/ruby-numo/gsl.git $ cd gsl $ rake build $ gem install pkg/numo-gsl-*.gem

rubygems is to be created.
